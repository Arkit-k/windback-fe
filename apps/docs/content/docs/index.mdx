# Welcome to Windback

Windback helps SaaS businesses detect subscription cancellations and recover churned customers using AI-generated personalized emails.

## What Windback Does

1. **Detects cancellations** — Receives webhooks from Stripe, Razorpay, and other payment providers when subscriptions are cancelled.
2. **Analyzes context** — Captures customer data, tenure, plan details, and cancellation reasons.
3. **Generates recovery emails** — AI creates 9 personalized email variants using different strategies (discounts, feature highlights, founder outreach, etc.).
4. **Sends and tracks** — Send emails with one click and track recovery outcomes.

## Key Features

- **Universal webhooks** — Stripe, Razorpay, Paddle, PayPal, and custom providers
- **AI recovery engine** — GPT-4o generates personalized winback emails
- **Cancellation widget** — Embed a lightweight JS widget to capture reasons
- **Recovery dashboard** — Track MRR at risk, recovery rate, and event outcomes
- **Developer-first** — RESTful API with SDKs for Node.js, Python, and Go
- **Resilient by design** — Webhooks never fail, widgets always work, even if your subscription expires

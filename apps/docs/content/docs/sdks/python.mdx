# Python SDK

<Callout type="warn">
The Python SDK is not yet published. Use the [REST API](/docs/api-reference) directly for now.
</Callout>

## Using the API directly

```python
import os
import requests

API_KEY = os.environ["WINDBACK_SECRET_KEY"]  # sk_...
BASE = "https://api.windbackai.com/api/v1"

headers = {"X-API-Key": API_KEY}

# Get stats
stats = requests.get(f"{BASE}/projects/my-app/stats", headers=headers).json()

# Track an event
requests.post(f"{BASE}/track", headers=headers, json={
    "customer_email": "user@example.com",
    "event": "login",
})

# List churn events
events = requests.get(f"{BASE}/projects/my-app/churn-events?limit=10", headers=headers).json()
```

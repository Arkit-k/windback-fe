# Python SDK

Official Python SDK for the Windback API.

## Installation

```bash
pip install windback
```

## Quick Start

```python
import os
from windback import Windback

client = Windback(api_key=os.environ["WINDBACK_SECRET_KEY"])  # sk_...

# Get stats
stats = client.stats.get()
print(f"Recovery rate: {stats.recovery_rate}%")

# List churn events
events = client.churn_events.list(status="new", limit=10)

# Generate recovery emails
variants = client.churn_events.generate_variants("event-id")

# Send a recovery email
client.churn_events.send_variant("event-id", "variant-id")

# Mark as recovered
client.churn_events.mark_recovered("event-id")
```

<Callout type="info">
The SDK uses your **secret API key** (`sk_...`) for authentication. Get your API key from **Settings > API Keys** in the [dashboard](https://windbackai.com/dashboard).
</Callout>

## Track Events

```python
# Track customer behavior for churn risk scoring
client.track(customer_email="user@example.com", event="login")

# With properties
client.track(
    customer_email="user@example.com",
    event="feature_used",
    properties={"feature": "export"},
)
```

## Configuration

```python
client = Windback(
    api_key=os.environ["WINDBACK_SECRET_KEY"],
    base_url="https://api.windbackai.com",  # optional
    timeout=30,  # seconds, optional
)
```

## Async Support

```python
from windback import AsyncWindback

client = AsyncWindback(api_key=os.environ["WINDBACK_SECRET_KEY"])

stats = await client.stats.get()
events = await client.churn_events.list()
```

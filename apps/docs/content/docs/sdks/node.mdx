# Node.js SDK

<Callout type="warn">
The Node.js SDK is not yet published. Use the [REST API](/docs/api-reference) directly for now.
</Callout>

## Using the API directly

```typescript
const API_KEY = process.env.WINDBACK_SECRET_KEY; // sk_...
const BASE = "https://api.windbackai.com/api/v1";

// Get stats
const stats = await fetch(`${BASE}/projects/my-app/stats`, {
  headers: { "X-API-Key": API_KEY },
}).then((r) => r.json());

// Track an event
await fetch(`${BASE}/track`, {
  method: "POST",
  headers: {
    "X-API-Key": API_KEY,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    customer_email: "user@example.com",
    event: "login",
  }),
});

// List churn events
const events = await fetch(`${BASE}/projects/my-app/churn-events?limit=10`, {
  headers: { "X-API-Key": API_KEY },
}).then((r) => r.json());
```
